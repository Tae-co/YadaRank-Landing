# Google Play Console 제출 가이드 — YadaRank

---

## 1. 개인정보처리방침

**입력할 URL:**
```
https://yadarank.com/privacy.html
```

---

## 2. 광고

### 현재 (광고 없음)
**선택:** `앱에 광고가 포함되어 있지 않습니다`

### 나중에 광고 추가 시
**선택:** `앱에 광고가 포함되어 있습니다` 로 변경

추가로 해야 할 작업:
- Google AdMob 등 광고 SDK 앱에 설치
- **광고 ID 섹션** (7번)도 `사용함` 으로 변경 필수
- 개인정보처리방침에 광고 관련 조항 추가 필요

---

## 3. 앱 액세스 권한

**선택:** `앱의 전체 또는 일부 기능이 제한됨`

그 다음 `새 안내 추가` 클릭:

| 항목 | 입력값 |
|------|--------|
| 안내 이름 | Google Login Test Account |
| 사용자 이름 (이메일) | _(테스트용 Gmail 주소)_ |
| 비밀번호 | _(테스트용 Gmail 비밀번호)_ |
| 기타 안내 | Tap "Google로 로그인" and sign in with the provided credentials. All features are accessible after login. |

> ⚠️ 테스트 계정은 2단계 인증 OFF 상태여야 함

---

## 4. 콘텐츠 등급

**절차:**
1. `등급 설문지 시작` 클릭
2. 앱 카테고리: `유틸리티 / 생산성 / 기타`
3. 아래 항목 모두 `아니요` 선택:
   - 폭력적인 콘텐츠
   - 성적인 콘텐츠
   - 도박
   - 사용자 생성 콘텐츠 공유
   - 위치 공유
   - 개인 정보 공유
4. 제출 → 예상 등급: **Everyone (전체 이용가)**

---

## 5. 타겟층 및 콘텐츠

| 항목 | 선택 |
|------|------|
| 주요 타겟 연령 | **13세 이상** |
| 어린이가 타겟에 포함되는가 | 아니요 |
| 앱 내 어린이 관련 콘텐츠 | 없음 |

---

## 6. 데이터 보안

### 수집하는 데이터 (모두 선택)

| 데이터 유형 | 항목 | 목적 |
|------------|------|------|
| 개인 정보 | 이름, 이메일 주소, 사진 | 계정 관리 |
| 앱 활동 | 앱 내 상호작용 (게임 기록, 랭킹) | 앱 기능 |
| 앱 정보 및 성능 | 앱 내 사용자 ID | 계정 식별 |

### 공통 답변

| 질문 | 답변 |
|------|------|
| 제3자와 데이터 공유 여부 | **아니요** |
| 데이터 암호화 전송 여부 | **예** (HTTPS) |
| 사용자가 삭제 요청 가능 여부 | **예** |
| 필수 데이터 수집 여부 | **예** (로그인 필수) |

### 나중에 광고/결제 추가 시 추가 선언 필요
| 데이터 유형 | 항목 | 추가 시점 |
|------------|------|----------|
| 기기 또는 기타 ID | 광고 ID | 광고 SDK 추가 시 |
| 금융 정보 | 구매 내역 | 인앱 결제/구독 추가 시 |

---

## 7. 광고 ID

### 현재 (광고 없음)
**선택:** `이 앱은 광고 ID를 사용하지 않습니다`

### 나중에 광고 추가 시
**선택:** `이 앱은 광고 ID를 사용합니다` 로 변경

> Android 13 이상 타겟팅 시 `AD_ID` 권한을 AndroidManifest.xml에 추가해야 함
> ```xml
> <uses-permission android:name="com.google.android.gms.permission.AD_ID"/>
> ```

---

## 8. 금융 기능

### 현재 (결제 없음)
**선택:** `해당 없음`

### 나중에 인앱 결제 / 구독 추가 시
**선택 변경:** `앱에서 구매가 발생합니다` 또는 `앱에서 구독이 발생합니다`

추가로 해야 할 작업:
- Google Play Billing Library 앱에 연동 (Capacitor 플러그인: `@capacitor-community/in-app-purchases` 또는 `cordova-plugin-purchase`)
- Google Play Console → **수익 창출 → 인앱 상품 / 구독** 에서 상품 등록
- 구독 상품 예시:
  - `yadarank_premium_monthly` — 월 구독
  - `yadarank_premium_yearly` — 연 구독
- 개인정보처리방침에 결제 관련 조항 추가 필요
- 구독 취소 방법 안내 문구 앱 내 필수 포함

> ⚠️ 인앱 결제는 Google이 30% 수수료 가져감 (연 매출 $1M 이하는 15%)
